# Passo a passo front - report

- Clonar repositório
  >
        git clone https://github.com/rothink/tj-te-back.git


- Entrar no repositório clonado
  >
        cd tj-te-back


- Subir os serviços com docker
  >
        docker-compose up --build -d


- go to page
  >
        http://172.23.0.2:3000/
